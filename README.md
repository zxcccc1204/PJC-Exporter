# PJC-Exporter
Export Timestamp of ClickUp into Excel.

# 📦 PJC Exporter 工時匯出工具

PJC Exporter 是一個用於每日自動化匯出 ClickUp 工時紀錄的工具，支援 GUI 操作與 Log 驗證，可快速產出 CSV 檔並檢查常見的填寫錯誤。  
適用於團隊內部每日工時稽核與報表作業流程。

---

## 🔧 功能特色

- 📆 GUI 日期選擇器，自訂匯出區間
- 📤 自動產出每日工時報表（CSV）
- ✅ 五大項目錯誤檢查（假別、工時、單位等）
- 🔒 支援 ClickUp API 與 Google Sheet 整合
- 🖥️ 可打包為 Windows 執行檔（.exe）
